pub struct WmNormalHints { /* private fields */ }
Expand description

Client requested hints about window geometry.

See the ICCCM spec for further details or the Xlib manual for more details of the data fromat but note that Penrose does not honour the following hints:

  • gravity
  • increment
  • aspect ratio
